The Wrestlers

Synchronous Smasher

The Hero of sequential, predictable functions.

Key Concepts:

  • Sequential execution
  • Predictable code flow
  • Blocking operations

function syncMatch() {
  console.log("Synchronous Smasher enters the ring.");
  console.log("Synchronous Smasher throws a punch!");
  console.log("Synchronous Smasher locks in a grapple!");
  console.log("Synchronous Smasher slams the opponent!");
  console.log("Synchronous Smasher leaves the ring victorious.");
}

syncMatch();

Asynchronous Assassin

The Villain who breaks the orderly flow.

Key Concepts:

  • Non-blocking operations
  • Callback functions
  • Event loop

function asyncMatch() {
  console.log("Asynchronous Assassin enters the ring!");

  setTimeout(() => {
    console.log("Asynchronous Assassin strikes after a 2-second delay!");
  }, 2000);

  console.log("Asynchronous Assassin distracts with a quick jab!");
}

asyncMatch();

Promise Enforcer

The Arbiter of Fate, guaranteeing resolution.

Key Concepts:

  • Promises
  • Resolve and reject
  • Chaining with .then() and .catch()

let promiseEnforcer = new Promise((resolve, reject) => {
  console.log("Promise Enforcer enters the ring.");

  let matchWon = true;  // Simulate outcome

  setTimeout(() => {
    if (matchWon) {
      resolve("Promise Enforcer wins the match!");
    } else {
      reject("Promise Enforcer is defeated...");
    }
  }, 3000);
});

promiseEnforcer
  .then(result => console.log(result))
  .catch(error => console.log(error));

Await Referee

The Guardian of Justice, ensuring fair play.

Key Concepts:

  • Async/await syntax
  • Error handling with try/catch
  • Sequential asynchronous operations

async function refereeMatch() {
  console.log("Await Referee enters the ring, watching Promise Enforcer...");

  try {
    const result = await promiseEnforcer;
    console.log("Await Referee declares: " + result);
  } catch (error) {
    console.log("Await Referee declares: " + error);
  }
}

refereeMatch();

About

Inspired by Roland Barthes' "Mythologies" and the spectacle of WWE wrestling, this tutorial transforms JavaScript concepts into a high-flying wrestling match. By mapping wrestling archetypes to programming functions, we create a storyline where justice in code is served, just like in the wrestling ring.
